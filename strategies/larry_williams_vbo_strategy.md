🦅 래리 윌리엄스 변동성 돌파 (Larry Williams Volatility Breakout, VBO)
핵심 철학: "어제의 변동 폭을 오늘 강하게 뚫어낸다면, 그 방향으로 에너지가 흐른다. 잡음(Noise)을 이기는 강력한 수급에 올라타라."

1. 🚀 페이즈 1: 대상 선정 (Setup)
전일 에너지를 응축했던 종목 중, 오늘 실시간으로 돈이 몰리는 '깡패 종목'만 감시망에 둡니다.

WatchList: OneilUniverseService의 Pool B (당일 실시간 거래대금 급등주 30종목) 만을 사용. Pool A(전일 기준 우량주)는 본 전략에서 사용하지 않음.

유동성 필터: 5일 평균 거래대금 100억 원 이상. (슬리피지/체결 실패 방지)

규모 필터: 시가총액 2,000억 원 이상. (잡주/이상급등 제외)

변동성 측정: 장 시작 전 1회, 전일 일봉으로 Range 계산 후 종목별 캐시에 보관.
Range = Yesterday_High − Yesterday_Low

2. 🎯 페이즈 2: 돌파 타격 (Trigger)
단순히 가격만 터치한다고 사지 않습니다. 돌파하는 순간의 '수급의 질'을 3중으로 검증합니다.

진입가 결정 (Target):
Target = Today_Open + (Range × K), 기본값 K = 0.5
(K는 Config 파라미터로 노출하여 0.3~0.7 범위에서 런타임 조정 가능)

스냅샷 체결강도 필터: 현재가가 Target을 돌파한 순간 현재가 API 1회 호출 → 당일 누적 체결강도 120% 이상일 것. (1초 단위 실시간 탐지는 사용하지 않음 — 스케줄러 부하 방지)

프로그램 수급 필터 (AND 조건):
- 당일 누적 프로그램 순매수 금액 ≥ 당일 거래대금 × 10%
- 당일 누적 프로그램 순매수 금액 양수(+)
(두 조건을 동시에 충족해야 함. 시총 대비 비율 조건은 사용하지 않음)

시간대 제한 (Intraday Window): 에너지가 집중되는 09:10 ~ 14:00 사이에만 신규 진입.
- 09:10 이전: 시가 확정 전 + 장 초반 휩소(개미 꼬시기) 회피
- 14:00 이후: 장 막판 돌파는 세력의 털기 가능성 + 오버나이트 배제 원칙상 신뢰도 급감

3. ✂️ 페이즈 3: 기계적 청산 (Exit)
래리 윌리엄스 전략의 핵심은 '오버나이트 배제'. 수익이든 손실이든 당일 장 안에서 마감합니다.

종가 강제 청산 (EOD Flatten): 15:20에 수익/손실과 무관하게 전량 시장가 매도.

방어적 칼손절: 진입가 대비 −3% 도달 시 장중이라도 즉시 전량 시장가 매도.

오버나이트 방어 (Safety Valve): 봇 재기동 등으로 전일 자정을 넘긴 포지션이 발견되면, 당일 장 시작 후 즉시 시장가 청산. (매수 체결 일자 ≠ 오늘이면 무조건 SELL)

4. ⚙️ 파라미터 요약
| 파라미터 | 기본값 | 설명 |
|---|---|---|
| `k_value` | 0.5 | Range 승수 (0.3~0.7 권장) |
| `min_5d_trading_value` | 100억 원 | 5일 평균 거래대금 하한 |
| `min_market_cap` | 2,000억 원 | 시가총액 하한 |
| `confidence_threshold` | 120% | 스냅샷 체결강도 하한 |
| `program_buy_ratio` | 10% | 프로그램 순매수 / 거래대금 하한 |
| `entry_start` | 09:10 | 신규 진입 시작 시각 |
| `entry_cutoff` | 14:00 | 신규 진입 마감 시각 |
| `stop_loss_pct` | −3.0% | 칼손절 기준 |
| `eod_flatten_time` | 15:20 | 종가 강제 청산 시각 |

5. 🔒 전략 불변 원칙 (Invariants)
- 오버나이트 금지: 당일 진입 → 당일 청산. 예외 없음.
- 1종목 1회 진입: 당일 동일 종목 재진입 금지 (`_bought_today` 세트로 관리).
- 포지션 사이즈: `base_strategy_config`의 `use_fixed_qty` 규칙을 상속.
- 하락장 가드: 코스닥 지수 20일선 우하향 시 신규 진입 중단 (후속 확장 포인트 — 본 버전에서는 포함하지 않음).
